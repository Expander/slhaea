SLHAea

another SUSY Les Houches Accord input/output library

Introduction

SLHAea is an easy to use input/output C++ library for the SUSY Les Houches Accord (SLHA)[1-3].

[1] P. Skands and others, arXiv:hep-ph/0311123
[2] B. Allanach and others, arXiv:0801.0045 [hep-ph]
[3] http://home.fnal.gov/~skands/slha/

Documentation

The API documentation can be found here.

Dependencies

To use SLHAea only the C++ Standard Library and some headers from the Boost C++ Libraries are required.

Download

You can download SLHAea in either tar.gz or zip formats.

For development of SLHAea Git is used. The SLHAea repository can be inspected and browsed online at GitHub and it can be cloned by running: 

$ git clone git://github.com/fthomas/slhaea
